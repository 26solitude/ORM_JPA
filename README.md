- **ISBN**: 9788960777330

## 🌱 학습 내용
- **2장**: JPA 기본 개념과 설정
- **3장**: 엔티티 매핑과 관계 매핑
- **4장**: JPQL 및 쿼리 실행
- **5장**: 영속성 컨텍스트와 트랜잭션 관리

이후 챕터는 점진적으로 추가될 예정입니다.
